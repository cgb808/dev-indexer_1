# frontend/ React Dashboard

Vite + React application providing chat UI, metrics dashboard, persona selection, TTS/STT integration.

Build Workflow:
1. `npm install`
2. `npm run build`
3. Assets copied to `app/static/dashboard/` via `make dashboard-build`.
